# Josh Bailey · [@anarkiwi](https://github.com/anarkiwi)

Networks, radios, Tesla coils, and 8-bit sound. Wellington, NZ · [vandervecken.com](https://www.vandervecken.com)

Engineer/experimenter — 14 yrs at Google (network research/SDN), independent since 2019; consultant + hardware hacker. `Python · C · C++ · Go · 6502/68HC11 asm`

### Work
- **SDN/Networking** — founding contributor to [FAUCET](https://github.com/faucetsdn/faucet) (production OpenFlow controller); [dovesnap](https://github.com/anarkiwi/dovesnap) (Docker OVS plugin), [pipette](https://github.com/anarkiwi/pipette) (SDN/NFV coprocessor), [faucetconfrpc](https://github.com/anarkiwi/faucetconfrpc). FAUCET deployed/cited by [NATO STO coalition tactical networks](https://www.sto.nato.int/document/a-software-defined-networking-architecture-for-coalition-tactical-military-networks/) (ICMCIS 2025), [Sandia National Laboratories](https://conference.faucet.nz/assets/files/Timothy+Toole+-+FAUCET+at+Sandia+National+Laboratories.pdf) (2019), and [IQT Labs *Defensive 5G*](https://forum.defcon.org/filedata/fetch?id=243179&d=1660004109) (DEF CON 30, 2022).
- **RF/SDR** — maintainer of [gamutRF](https://github.com/IQTLabs/gamutRF) (SDR scanner/collector); [samples2djidroneid](https://github.com/anarkiwi/samples2djidroneid) (DJI DroneID from I/Q), [mt_rx_samples_to_file](https://github.com/anarkiwi/mt_rx_samples_to_file).
- **ML** — [preframr](https://github.com/anarkiwi/preframr) (transformer that composes SID register streams), RF signal classification, [jetson-triton](https://github.com/anarkiwi/jetson-triton).
- **Embedded** — Arduino Due / SAMD21 (ARM Cortex-M0/M3) firmware, fixed-point DSP, real-time MIDI: [chime_red2](https://github.com/anarkiwi/chime_red2) (musical Tesla coil controller), [vessel](https://github.com/anarkiwi/vessel) (C64 MIDI interface), [vap](https://github.com/anarkiwi/vap) / [vmp](https://github.com/anarkiwi/vmp) (SID synth firmware).
- **Vector graphics** — Vectrex/PiTrex tooling: [vekterm](https://github.com/anarkiwi/vekterm) (baremetal PiTrex receiver that boots a Pi straight into drawing vectors on a real Vectrex), [pyvterm](https://github.com/anarkiwi/pyvterm) (drives a PiTrex/Vectrex over serial from Python).
- **C64 / chiptune** — [Vessel](https://github.com/anarkiwi/vessel) (high-performance C64 MIDI interface), [desidulate](https://github.com/anarkiwi/desidulate) (SID analysis), [asid-vice](https://github.com/anarkiwi/asid-vice) (VICE with ASID support).
- **Protocol analysis** — [Wireshark](https://www.wireshark.org) contributor (IPDC dissector, Lucent/Ascend trace parsing); [tshark2pandas](https://github.com/anarkiwi/tshark2pandas).
